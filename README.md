📘 Ticket Counter Simulation (Queue)
📌 Deskripsi

Program ini merupakan simulasi sistem antrian pada loket tiket menggunakan konsep Queue (FIFO). Penumpang akan masuk ke antrian, dilayani oleh agen, lalu keluar setelah selesai.

🎯 Tujuan
Mengimplementasikan Queue (enqueue & dequeue)
Memahami sistem antrian
Mensimulasikan proses pelayanan

⚙️ Fitur
Kedatangan penumpang secara acak
Proses antrian (enqueue & dequeue)
Pelayanan oleh agen
Perhitungan waktu tunggu

🧩 Method Utama
_handleArrival() → menangani penumpang masuk
_handleBeginService() → memulai pelayanan
_handleEndService() → mengakhiri pelayanan
